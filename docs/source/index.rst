Bem-vindo(a) a Documentação do PimentaDoc!
==========================================

O **PimentaDoc** é um gerenciador de *serviços planos* definidos pelo
próprio usuário no contexto do programa através de *phandlers*. Assim,
ele é perfeito para a criação de servidores multi serviço e *home
servers*.
